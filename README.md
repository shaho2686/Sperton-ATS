# Sperton Recruitment Portal

Full-stack recruitment portal built for assessment review.

## Quick Start

1. Install dependencies

```bash
npm install
```

2. Seed sample candidates

```bash
node scripts/seed-sample-candidates.js
```

3. Start the project

```bash
npm start
```

4. Open in browser

```text
http://localhost:3000/portal
```
